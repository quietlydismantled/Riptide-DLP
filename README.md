## Downloads

Pre-built binaries for Windows / Linux / macOS (x64 + ARM64) are on the [Releases page](https://github.com/quietlydismantled/riptide-dlp/releases).

## Build from source

```bash
git clone https://github.com/quietlydismantled/riptide-dlp.git
cd riptide-dlp
dotnet build RiptideDlp.sln
dotnet run --project src/RiptideDlp
```

Requires .NET 8 SDK.
